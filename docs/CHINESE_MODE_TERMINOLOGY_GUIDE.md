# 中文模式术语规范

## 目标

中文模式下，页面的导航、栏目名、按钮文案、说明文字应优先使用中文。  
品牌名和少数行业内约定俗成的技术术语可以保留英文，避免出现整页中文里夹杂大量无必要英文栏目名的情况。

## 必须保留英文

- 品牌名：`AI Gateway`
- 模型厂商名：`OpenAI`、`Anthropic`、`Gemini`、`DeepSeek`、`Mistral`
- 技术协议或生态名：`API`、`SDK`、`OAuth`、`Webhook`、`RBAC`
- 编程语言与运行时：`Python`、`Node.js`、`cURL`
- 行业内通用技术词：
  - `Token`
  - `Fallback`
  - `OpenAI Compatible`
  - `Prompt`

## 应该中文化

- 页面导航：`Home`、`Docs`、`Pricing`、`Contact`
- 页面栏目名：`Launch Desk`、`Catalog`、`Query Desk`、`Model Rates`
- 功能面板名：`Session Snapshot`、`Control Deck`、`Workbench`
- 说明型标签：`Provider`、`Input`、`Output`、`Accepted Rails`
- 营销型短语：`Start Here`、`Promise`、`Fast Onboarding`

## 推荐写法

- `Playground`
  - 页面导航和按钮文案中文化为 `试验台`
  - 在技术说明里可以保留 `Playground`
- `OpenAI Compatible`
  - 推荐写成 `OpenAI 兼容`
- `Token`
  - 推荐直接保留 `Token`
- `System Prompt`
  - 推荐保留 `System Prompt`
- `Max Tokens`
  - 推荐保留 `Max Tokens`
